Draw a flowchart for a user authentication process. The user enters their email and password. The system checks if the account exists — if not, show "Account not found" error. If it exists, validate the password. On wrong password, increment a failed attempt counter. After 3 failed attempts, lock the account and send a reset email. On correct password, check if MFA is enabled — if yes, send an OTP and wait for verification; if no, redirect straight to the dashboard. Successful MFA verification also goes to dashboard.
