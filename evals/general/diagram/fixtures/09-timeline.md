Draw a timeline of major web platform milestones: 1991 — World Wide Web invented by Tim Berners-Lee (HTTP/HTML). 1994 — Netscape Navigator released, W3C founded. 1995 — JavaScript created by Brendan Eich, PHP released. 1996 — CSS1 published by W3C, Flash introduced. 1998 — Google founded, XML 1.0 specification. 2004 — Firefox 1.0 released, Web 2.0 term coined. 2007 — iPhone launched with Mobile Safari, WebKit open-sourced. 2008 — Google Chrome released, HTML5 draft published. 2014 — HTML5 finalized as W3C Recommendation. 2015 — HTTP/2 published, ES6/ES2015 JavaScript. 2017 — WebAssembly MVP shipped in all major browsers. 2022 — HTTP/3 (QUIC) standardized.
