Create an ER diagram for a blog platform. Entities: User (id PK, email UK, username UK, role), Post (id PK, title, slug UK, body, status, published_at), Comment (id PK, body, created_at), Tag (id PK, name UK, slug UK), Category (id PK, name UK). Relationships: User writes many Posts, Post belongs to one User. User writes many Comments, Comment belongs to one User. Post has many Comments. Post tagged with many Tags (and Tags can appear on many Posts). Post belongs to one Category, Category has many Posts.
