Draw a cloud architecture diagram for our AWS deployment. There's a Client (browser) that hits CloudFront CDN. CloudFront serves static assets from an S3 bucket and forwards API requests to an Application Load Balancer. The ALB routes to ECS containers running the Node.js API. The API connects to an RDS Aurora PostgreSQL database and an ElastiCache Redis cluster (for sessions/caching). The API also sends messages to an SQS queue, which triggers Lambda functions for async processing. Lambda reads/writes to S3 for file processing. Group the ECS, RDS, ElastiCache, SQS, Lambda into a VPC boundary.
